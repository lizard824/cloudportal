Overview

What is IaaS?

Infrastructure as a service (IaaS) is an instant computing infrastructure, provisioned and managed over the Internet. Quickly scale up and down with demand, and pay only for what you use.

IaaS helps you avoid the expense and complexity of buying and managing your own physical servers and other datacenter infrastructure. Each resource is offered as a separate service component, and you only need to rent a particular one for as long as you need it. The cloud computing service provider manages the infrastructure, while you purchase, install, configure, and manage your own software—operating systems, middleware, and applications.

IaaS provides not only simple infrastructure resources but also assorted solution on infrastructure specific to different application features, 7*24 supports on server maintenance, highly secured storage and backup approach, and customized services.

Advantages of IaaS

Innovate rapidly. As soon as you’ve decided to launch a new product or initiative, the necessary computing infrastructure can be ready in minutes or hours, rather than the days or weeks—and sometimes months—it could take to set up internally.IaaS provides not only simple infrastructure resources but also assorted solution on infrastructure specific to different application features, 7*24 supports on server maintenance, highly secured storage and backup approach, and customized services.

Better connectivity with business within Lenovo. IaaS provides different zones and locations to divide different Internet connection requirements.

Increase supportability and disaster recovery. With IaaS you will enjoy 24x7x365 Support and high-performance storage and back up which you will not be troubled by loosing data.

Eliminates expense and reduces ongoing cost. IaaS sidesteps the upfront expense of setting up and managing an on-site datacenter, making it an economical option for start-ups and businesses testing new ideas.

Focus on your core business. IaaS frees up your team to focus on your organization’s core business rather than on IT infrastructure.

Common IaaS business scenarios

Typical things businesses do with IaaS include:

Test and development.  Teams can quickly set up and dismantle test and development environments, bringing new applications to market faster. IaaS makes it quick and economical to scale up dev-test environments up and down.

Website hosting.   Running websites using IaaS can be less expensive than traditional web hosting.

Storage, backup, and recovery.   Organizations avoid the capital outlay for storage and complexity of storage management, which typically requires a skilled staff to manage data and meet legal and compliance requirements. IaaS is useful for handling unpredictable demand and steadily growing storage needs. It can also simplify planning and management of backup and recovery systems.

Web apps.   IaaS provides all the infrastructure to support web apps, including storage, web and application servers, and networking resources. Organizations can quickly deploy web apps on IaaS and easily scale infrastructure up and down when demand for the apps is unpredictable.

High-performance computing.    High-performance computing (HPC) on supercomputers, computer grids, or computer clusters helps solve complex problems involving millions of variables or calculations. Examples include earthquake and protein folding simulations, climate and weather predictions, financial modeling, and evaluating product designs.

Big data analysis.    Big data is a popular term for massive data sets that contain potentially valuable patterns, trends, and associations. Mining data sets to locate or tease out these hidden patterns requires a huge amount of processing power, which IaaS economically provides.

IaaS core services

Core Services include server maintenance and troubleshooting, global AD management, global DNS management, global ID management and backup service. Server maintenance includes security baseline configuration, resizing, patching and so on.

Delivered by Instances

IaaS provides one-stop access to allow you to apply computing and storage resources delivered by per instance and track the status of your application at any time. IaaS simple web service interface allows you to obtain and configure capacity with minimal friction. As soon as you get the instance, you can begin to set up your environment immediately without any concern of the infrastructure and maintenance of the Operating System. Therefore, you could avoid the need to provision expensive IT infrastructure and hire large network teams.

Physical Machine

Physical Machines are commonly offered to computationally intensive required applications. Compared to Virtual Machines, Physical Machines provide more stable and better performance environment as you are the only user of the hardware resources. However, Physical Machine cost relatively higher due to the hardware cost.

Virtual Machine

There are currently mainly two platforms delivering virtual machines. One is implemented by VMware which is a mature virtualization commercial product. The other one is open source Openstack platform implemented by KVM. Openstack is a software platform for cloud computing. With high price of VMware license, Openstack are more and more popular for its usability and flexibility.

Delivered by Tenants

IaaS also can deliver a certain amount of computing, storage and even network resources in the form of tenants. Being an owner of a tenant, you have complete control of your computing and storage resources. You can deploy instances with just a few clicks via a clean and easy-to-use console. It reduces the time required to obtain and boot new server instances to minutes, allowing you to quickly scale capacity, both up and down, as your computing requirements change.

IaaS advanced services

Beside core services, IaaS also provides customized services on infrastructure such as Load Balancer as a Service (LBaaS), image management, monitor and so on. Furthermore, more and more innovation applications are developing on platforms based on customized infrastructure facilities with specific characteristics, such as Big Data and E-commerce.

Object Storage Service

LBaaS (Load-Balancing-as-a-Service) is an advanced service of Network. It allows for proprietary and open-source load balancing technologies to drive the actual load balancing of requests. Thus, user can choose which back-end technology to use.

Cloud Monitor

Cloud Monitor Service is a flexible monitoring service that provides in-depth insights into cloud deployments. It provides advanced analytics on critical metrics such as CPU utilization, latency and also lets you customize metrics specific to business requirements.

Security Group

The default set of rules in place when a new tenant is created allows for any outbound communication, but no inbound. Security groups are the mechanism we can use to modify these rules.
