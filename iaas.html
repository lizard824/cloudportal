Overview

  • Infrastructure as a Service (IaaS) is a platform that provides variety kinds of virtualized and physical infrastructure resources and solution above, according to different application features and requirements.
  • IaaS is capable of fast delivery and quick response to business, 7*24 supports on server maintenance, highly secured storage and backup approach, and customized services.

Infrastructure Resources

Infrastructure resources include compute resources such as Virtual Machines which are provided based on VMware, Openstack, Hyper-v and Power platforms, and X86/Power Physical Machines with Operating systems (Linux, Windows and AIX). The facilities also include storage resources which can be provided according to different data type.

IaaS core services

  • Core Services include server maintenance and troubleshooting, global AD management, global DNS management, global ID management and backup service.
  • Server maintenance includes security baseline configuration, resizing, patching and so on.
  • Backup provisions include database backup which define backup schedule, add backup node and backup selection by customer backup request to implement backup for database, and file backup which define backup schedule, add backup node and backup selection by customer backup request to implement backup for file.

IaaS advanced services

Beside core services, IaaS also provides customized services on infrastructure such as Load Balancer as a Service (LBaaS), image management, monitor and so on. Furthermore, more and more innovation applications are developing on platforms based on customized infrastructure facilities with specific characteristics, such as Big Data and DTC.
