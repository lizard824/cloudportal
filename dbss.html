Overview

DB Self-Servcie (DBSS) is handling daily frequent DB requests automatically, including new database, new account , add privileges ,reset password, search blocked process and so on. It can not only improve work efficiency, but also avoid manual mistakes .

Services Catalog

DBSS provides service lists as below including SQL Server , MySQL , Oracle ,PostgreSQL, DB2.

New Database

If want to new a database , please choose this feature.

New Account

If want to new a database account with needed privileges , please choose this feature.

Add Privileges

If want to add privileges for an exited database account , please choose this feature.

Reset Password

If forget the database account’s password, please choose this feature to reset password.

Search Blocked Process

If the performance of DB is poor, please choose this feature firstly to check the status of process. If blocked process exits, this feature can help check the detail information and kill the blocked process.

Deploy Go-live

If want to go-live sql scripts , please choose this feature.
